# TextDiff — Визуальное сравнение текстов и файлов

Кроссплатформенное desktop-приложение (macOS + Windows) для визуального сравнения двух версий текста или файла. Аналог Meld с современным интерфейсом.

---

## Что умеет приложение (MVP)

- **Режим «Текст»** — вставить два текста и сразу увидеть diff
- **Режим «Файлы»** — открыть два файла с диска
- **Drag & Drop** — перетащить файл прямо на панель
- Подсветка изменений: добавленное (зелёное), удалённое (красное), изменённое (жёлтое)
- Подсветка изменений **внутри строки** по словам
- Навигация по блокам изменений: **F7 / Shift+F7**
- Редактирование правой панели прямо в редакторе
- Сохранение результата в файл
- Светлая / тёмная тема
- Игнорирование пробелов, перенос строк, режим "рядом" или "вместе"
- Автоопределение языка (JS, TS, Python, Ruby, JSON, Markdown и др.)

## Установка — пошаговая инструкция

### Шаг 1 — Скачайте проект

Разместите папку `textdiff` в любом месте на вашем компьютере.

---

### Шаг 2 — Установите Node.js

1. Откройте https://nodejs.org
2. Нажмите **Download Node.js (LTS)**
3. Запустите установщик и следуйте инструкциям
4. После установки перезапустите терминал

---

### Шаг 3 — Установите Rust

**macOS / Linux:**
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
После установки закройте и откройте терминал снова.

**Windows:**
1. Откройте https://rustup.rs
2. Скачайте и запустите `rustup-init.exe`
3. Следуйте инструкциям (нажимайте Enter для стандартных значений)
4. Перезапустите терминал

---

### Шаг 4 — Установите зависимости системы

**macOS:**
```bash
xcode-select --install
```
(Если появится диалог — нажмите «Установить»)

**Windows:**
Дополнительных шагов нет.

---

### Шаг 5 — Запустите скрипт установки

**macOS / Linux** — откройте Терминал в папке проекта:
```bash
bash setup.sh
```

**Windows** — дважды кликните `setup.bat` или откройте в папке:
```
setup.bat
```

Скрипт сам проверит все зависимости и установит npm пакеты.

---

### Шаг 6 — Запустите приложение

Откройте терминал в папке `textdiff` и выполните:
```bash
npm run tauri dev
```

Через ~30–60 секунд откроется окно приложения.

---

### Шаг 7 — Собрать финальный установщик (опционально)

```bash
npm run tauri build
```

Результат будет в папке:
- `src-tauri/target/release/bundle/dmg/` — macOS (.dmg)
- `src-tauri/target/release/bundle/msi/` — Windows (.msi)
- `src-tauri/target/release/bundle/nsis/` — Windows (.exe)

## Использование

### Сравнение текстов (режим «Текст»)
1. Вставьте исходный текст в левое поле (Оригинал)
2. Вставьте изменённый текст в правое поле (Изменённый)
3. Diff появляется автоматически в редакторе ниже

### Сравнение файлов (режим «Файлы»)
1. Нажмите **📁 Оригинал** → выберите первый файл
2. Нажмите **📁 Изменённый** → выберите второй файл

### Навигация по изменениям
- Кнопки **▲ Пред.** / **▼ След.** в тулбаре
- Клавиши **F7** (следующее) / **Shift+F7** (предыдущее)

### Сохранение
- Нажмите **💾 Сохранить** — откроется диалог сохранения

## Если что-то пошло не так

**`npm run tauri dev` падает с ошибкой Rust:**
```bash
rustup update
```

**Monaco не грузится:**
```bash
rm -rf node_modules
npm install
```

**На Windows ошибка WebView2:**
Установите Microsoft Edge WebView2 Runtime:
https://developer.microsoft.com/en-us/microsoft-edge/webview2/
