## Приоритеты: с чего начать

### 1. Разблокировать сборку (сейчас)
```bash
npm run tauri icon path/to/icon-1024x1024.png
```
Папка `src-tauri/icons/` пустая — без иконок `npm run tauri build` упадёт с ошибкой.

### 2. Первые три доработки (высокий эффект, мало кода)

**Индекс изменения «3 из 12»** — одна строка в `onDidUpdateDiff`, данные уже есть в `navRef`.

**Сохранение настроек** — добавить `tauri-plugin-store`, чтобы тема и опции не сбрасывались при перезапуске.

**Кнопки «принять блок»** — accept left → right и reject. Это шаг от просмотрщика к инструменту редактирования (Этап 3 по ТЗ).

### 3. UX-улучшения (средний приоритет)
- Collapse unchanged sections
- Переключатель sync scroll
- Кнопка «Сохранить как» отдельно от «Сохранить»
